# Beans Coffee - Animated Business Card

Animowana wizytówka kawiarni Beans Coffee - Kawiarnia przyjazna zwierzętom.
Magda Zajdlic

Format: 1920x1080px

## 🚀 Deployment na GitHub Pages

### Krok 1: Push kodu do repozytorium
```bash
cd beans-coffee-export
git init
git add .
git commit -m "Initial commit - Beans Coffee animation"
git branch -M main
git remote add origin https://github.com/antekku9/Testt.git
git push -u origin main
```

### Krok 2: Włącz GitHub Pages
1. Przejdź do Settings → Pages w repozytorium
2. W sekcji "Source" wybierz **GitHub Actions**
3. GitHub automatycznie wykryje workflow i rozpocznie deployment

### Krok 3: Gotowe!
Animacja będzie dostępna pod adresem:
```
https://antekku9.github.io/Testt/
```

## 💻 Lokalne uruchomienie

```bash
pnpm install
pnpm dev
```

## 📦 Build

```bash
pnpm build
```

Build zostanie utworzony w katalogu `dist/`.

## ✨ Efekty w animacji

- Unoszące się ziarna kawy
- Efekt pary z kawy
- Płynne gradienty w odcieniach kawy
- Latające ikony filiżanek w tle
- Dekoracyjna fala na górze
- Pulsujące kropki na dole
- Automatyczne powtarzanie animacji
- Stopniowe pojawianie się elementów
- Format Full HD (1920x1080)
